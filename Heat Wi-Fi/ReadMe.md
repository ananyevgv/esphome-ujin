## 📌 Информация об устройстве

| Параметр | Значение |
|----------|----------|
| **Чип** | ESP8266 |
| **Модель** | Heat Wi-Fi |
| **Назначение** | Термостат для управления нагревательными приборами |

---

## 🧩 Аппаратная комплектация

| Компонент | Описание |
|-----------|----------|
| 🌡️ Датчик T/H | **SHT30** (температура и влажность) |
| 💡 Светодиоды | Адресные светодиоды **WS2812** |
| 👆 Сенсорные кнопки | **2 шт.** (через емкостной контроллер **CAP1293**) |
| 🔊 Зуммер | ✅ |
| 🌡️ Внешний датчик | Вход для подключения NTC-термистора |
| ⚡ Реле | Управление нагрузкой **220В**, до **8А** |

---

## ✅ Реализованные функции

| № | Функция |
|:--:|---------|
| 1 | Работа **термостата** (поддержание заданной температуры) |
| 2 | Цветовая индикация режима работы (WS2812) |
| 3 | Отображение установленной температуры нагрева |
| 4 | Измерение температуры и влажности (SHT30) |
| 5 | Подключение внешнего NTC-датчика |
| 6 | Сенсорное управление (CAP1293) |
| 7 | Звуковое оповещение (зуммер) |

## 📊 Спецификация

| Параметр | Значение |
|----------|----------|
| MCU | ESP8266 |
| Датчик T/H | SHT30 |
| Внешний датчик | NTC-термистор |
| Сенсорный контроллер | CAP1293 |
| Сенсорные кнопки | 2 |
| Индикация | WS2812 (адресная RGB) |
| Реле | 220В, до 8А |
| Оповещение | зуммер |
| Wi-Fi | ✅ |

---

## 🔧 Принцип работы

1. Термостат измеряет температуру (встроенный SHT30 или внешний NTC)
2. Пользователь задаёт желаемую температуру через сенсорные кнопки
3. WS2812 индицирует режим работы цветом
4. Реле включает/выключает нагрузку (220В, до 8А) для поддержания температуры
5. При срабатывании возможно звуковое оповещение (зуммер)

---

## 🎛️ Сенсорное управление

| Особенность | Описание |
|-------------|----------|
| Контроллер | CAP1293 (ёмкостной сенсорный контроллер) |
| Кнопки | 2 шт. |
| Преимущество | Стабильная работа даже через толстый слой стекла/пластика |

---

## ⚠️ Важное примечание

> Внешний NTC-датчик подключается отдельно и позволяет измерять температуру в удалённой точке.
