  <h1>🔄 Альтернативная плата для Ujin EKF</h1>
  <p><strong>Замена платы для Aqua / Pulse / Heat с поддержкой 3 реле и 2 внешних устройств</strong></p>
  <br/>
</div>


\---

## 📌 Совместимость

Данная плата предназначена для замены штатной платы в устройствах:

|Устройство|Тип|
|-|-|
|💧 **Aqua EKF**|Контроллер протечки воды|
|⚡ **Pulse EKF**|Управление светом / диммер|
|🌡️ **Heat EKF**|Термостат|

\---

## 🧩 Возможности платы

|Параметр|Значение|
|-|-|
|**Реле**|**3 шт.** (IO33, IO25, IO27)|
|**Внешние устройства**|**2 шт.** (через R7, R8)|

\---

## 🔌 Подключение внешних устройств

В зависимости от типа подключаемого устройства, необходимо установить соответствующий резистор и использовать определённые пины:

|Тип устройства|Pins|Резистор|Действие|
|-|-|-|-|
|💧 **Проводной датчик протечки**|R8 (I38), R7 (I35)|100 Ом|установить **PR5V**|
|🌡️ **NTC датчик температуры**|R8 (I38), R7 (I35)|10 кОм|установить **PR3V**|
|🔘 **Внешний выключатель**|R8 (I38), R7 (I35)|10 кОм|установить **PR3V**|

\---

## ⚡ Реле

|Номер|GPIO|Назначение|
|:-:|:-:|-|
|**K1**|IO33|Реле 1|
|**K2**|IO25|Реле 2|
|**K3**|IO27|Реле 3|

## 📊 Схема подключения



┌─────────────────────────────────────────────┐

│ Альтернативная плата │

├─────────────────────────────────────────────┤

│ │

│ IO33 ─────────► Реле 1 (K1) │

│ IO25 ─────────► Реле 2 (K2) │

│ IO27 ─────────► Реле 3 (K3) │

│ │

│ I35 (R7) ─────┐ │

│ ├──► Внешнее устройство 1 │

│ I38 (R8) ─────┘ (датчик/кнопка/NTC) │

│ │

└─────────────────────────────────────────────┘

\## 🔧 Настройка резисторов



| Резистор | Номинал | Напряжение | Для каких устройств |

|:--------:|:-------:|:----------:|---------------------|

| \*\*PR5V\*\* | 100 Ом | 5В | Проводной датчик протечки |

| \*\*PR3V\*\* | 10 кОм | 3.3В | NTC датчик / внешний выключатель |



> ⚠️ \*\*Важно\*\*: Правильный выбор резистора \*\*обязателен\*\* для корректной работы подключаемого устройства!



\---



\## ⚠️ Важные примечания



| № | Примечание |

|:--:|------------|

| 1 | Плата полностью заменяет штатную в корпусах Aqua, Pulse, Heat |

| 2 | Все 3 реле могут использоваться независимо |

| 3 | Для каждого внешнего устройства настраивается свой резистор |

| 4 | Перед прошивкой \*\*проверьте правильность установки резисторов\*\* |



\---



\## 📦 Примеры использования



| Сценарий | Реле | Внешние устройства |

|----------|:----:|--------------------|

| Aqua (протечка) | K1 — кран | R7/R8 — датчики протечки |

| Pulse (свет) | K1, K2, K3 — каналы света | R7/R8 — внешние выключатели |

| Heat (отопление) | K1 — насос/клапан | R7 — NTC датчик |
