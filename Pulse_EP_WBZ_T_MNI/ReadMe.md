---

## 📌 Совместимость

Прошивка подходит для:

- **EP-WBZ-T-MNI**
- И аналогичных устройств на базе ESP32 PICO

---

## 🧩 Аппаратная платформа

| Компонент | Описание |
|-----------|----------|
| **Чип** | ESP32 PICO |
| **Светодиоды** | 35 адресных светодиодов WS2812 |

---

## 🎮 Управление и интерфейсы

| Компонент | Описание |
|-----------|----------|
| Сенсорные кнопки | 2 шт. |
| Зуммер | ✅ |
| ZigBee | Контроллер **2652** |

---

## 📡 Датчики и сенсоры

| Датчик | Особенности |
|--------|-------------|
| Микрофоны | **2 шт.** |
| Датчик движения | С регулировкой дальности срабатывания через **I2C цифровой потенциометр** |
| Датчик температуры | ✅ |
| Датчик освещённости | ✅ |

---

## 📊 Краткая спецификация

| Параметр | Значение |
|----------|----------|
| Базовая платформа | ESP32 PICO |
| Адресные LED | 35 × WS2812 |
| Сенсорные кнопки | 2 |
| ZigBee | CC2652 |
| Микрофоны | 2 |
| Датчик движения | регулируемая дальность (I2C потенциометр) |
| Датчик температуры | ✅ |
| Датчик освещённости | ✅ |
| Зуммер | ✅ |

---

## 🔧 Особенности

- ⚙️ **Регулировка дальности датчика движения** — осуществляется через I2C цифровой потенциометр
- 🔊 **Два микрофона** — для улучшенного распознавания шума/звука
- 🎨 **35 адресных светодиодов** — гибкая настройка подсветки
