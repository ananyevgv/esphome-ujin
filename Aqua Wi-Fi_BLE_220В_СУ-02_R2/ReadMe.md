## 📊 Спецификация

| Параметр | Значение |
|----------|----------|
| MCU | ESP32 PICO |
| Датчики протечки | до 2 шт. (поддержка UAS-W) |
| Реле | 220В, до 5А |
| Управление | сенсорные кнопки (2 шт.) |
| Индикация | WS2812 (адресная RGB) |
| Оповещение | зуммер |
| BLE | ✅ |
| Wi-Fi | ✅ |

---

## 🔧 Принцип работы

1. Датчик протечки (проводной UAS-W) обнаруживает воду
2. Контроллер включает **световую индикацию** (WS2812)
3. Активируется **звуковое оповещение** (зуммер)
4. Реле **перекрывает кран** 220В
5. Через сенсорные кнопки или приложение можно сбросить аварию

---

## ⚠️ Важное примечание

> Для работы с беспроводным датчиком **UAS-B** используйте отдельный BLE-мост.  
> Данный контроллер подключает **только проводной датчик UAS-W**.
